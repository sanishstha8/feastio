# RestaurantOS 🍽️

A full-stack restaurant management system built with Django and JavaScript.

## Team Members
- **Person 1 Name** — Backend (Django REST API, database models, authentication)
- **Person 2 Name** — Frontend (JavaScript, staff management, menu)
- **Person 3 Name** — UI/UX (CSS styling, templates, documentation)

## Tech Stack
- **Backend:** Django 5, Django REST Framework, PostgreSQL
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Auth:** JWT (JSON Web Tokens)

## Features
- Role-based login (Manager, Waiter, Kitchen)
- Table and order management with real-time status
- Menu management with categories
- Staff management with shift tracking
- Sales reports with date filtering
- Auto-refresh every 30 seconds
